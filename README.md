# 考研智能错题本

一个移动端优先的考研错题管理工具，适合手机、平板和电脑使用。第一版不依赖后端，数据保存在浏览器 `localStorage` 中。

## 功能

- 录入错题：科目、知识点、错因、来源、题目描述、错误思路、正确思路
- 题目照片：支持从手机或平板相册选择图片并保存在本地浏览器中
- 今日复习：根据下一次复习日期自动生成清单
- 错题列表：支持科目、错因和关键词筛选
- 统计分析：自动统计高频错因和高频知识点
- AI 答疑：支持配置 OpenAI-compatible API Base URL、API Key 和模型名进行聊天答疑
- PWA 支持：部署后可添加到手机或平板主屏幕

## 使用方式

直接打开 `index.html` 即可查看原型。

如果希望 PWA 缓存和安装能力正常工作，建议用本地服务或部署到 Vercel/GitHub Pages。
